# Ready to Eat - Meal Delivery Landing Page

A vibrant, high-converting landing page built for a modern meal prep and delivery service, designed to showcase chef-prepared meals, flexible subscription plans, and customer testimonials.

## 🎯 Why I Built This
I created this project to focus heavily on marketing-driven web design, user experience (UX) layout patterns, and advanced CSS styling. The primary goal was to handle complex modern layouts—like pricing tables and step-by-step feature guides—while maintaining perfect visual hierarchy and responsive behavior.

## ✨ Key Features
* 🍲 **Visual Menu Showcase:** A clean card-based layout featuring appetizing meal previews, dietary tags (e.g., Vegan, Keto), and calorie counts.
* 💳 **Interactive Subscription Tiers:** A structured pricing section that clearly differentiates meal plans, making it easy for users to choose a package.
* 📈 **Conversion-Optimized Hero Section:** A compelling headline paired with an immediate "View Plans" Call-to-Action (CTA) to maximize user sign-ups.
* 📱 **Seamless Mobile Layout:** Fully optimized for busy users ordering food on the go from their smartphones or tablets.

## 🛠️ Tech Used
* Semantic HTML5 (Sections, Articles, Nav)
* Custom CSS3 (Flexbox & CSS Grid for card layouts)
* Advanced CSS Effects (Hover animations, transitions, and shadow layers)
* Responsive Web Design (Media Queries)
