# Ready to Eat - Restaurant Landing Page

A vibrant, modern landing page built for a contemporary restaurant, designed to showcase the culinary menu, restaurant atmosphere, and drive customer table reservations.

## 🎯 Why I Built This
I created this project to focus heavily on local business web design, user experience (UX) layout patterns, and advanced CSS styling. The primary goal was to handle complex modern layouts—like multi-category food menus and step-by-step reservation flows—while maintaining perfect visual hierarchy and responsive behavior.

## ✨ Key Features
* 📜 **Digital Food Menu:** A beautifully structured card layout featuring appetizing dish descriptions, dietary tags (e.g., Gluten-Free, Vegan), and clear pricing.
* 📅 **Reservation Call-to-Action:** A strategically placed, frictionless booking section designed to maximize evening and weekend table reservations.
* 📸 **Atmosphere & Gallery Showcase:** High-impact visual sections highlighting the restaurant's interior aesthetic and signature chef creations.
* 📱 **Seamless Mobile Layout:** Fully optimized for guests looking up menu items, hours, or locations on the go from their smartphones.

## 🛠️ Tech Used
* Semantic HTML5 (Sections, Articles, Nav)
* Custom CSS3 (Flexbox & CSS Grid for card layouts)
* Advanced CSS Effects (Hover animations, transitions, and image overlays)
* Responsive Web Design (Media Queries)
